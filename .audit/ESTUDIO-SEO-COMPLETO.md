### 4.4 Performance y Core Web Vitals

| Métrica | Objetivo | Estado Estimado |
|---------|----------|-----------------|
| LCP (Largest Contentful Paint) | < 2.5s | ~3s (Mejorable) |
| FID (First Input Delay) | < 100ms | ~80ms (OK) |
| CLS (Cumulative Layout Shift) | < 0.1 | ~0.15 (Mejorable) |

**Problemas de Performance:**
1. Google Fonts sin `font-display: swap`
2. CSS no crítico no diferido
3. Imágenes externas (Unsplash) sin control de caché
4. Sin preload de recursos críticos

**Soluciones:**
```html
<!-- Optimizar Google Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Poppins:wght@600;700&display=swap" rel="stylesheet">

<!-- Preload recursos críticos -->
<link rel="preload" href="/css/style.css" as="style">
<link rel="preload" href="/js/app.js" as="script">

<!-- Preconnect a dominios externos -->
<link rel="preconnect" href="https://images.unsplash.com">
<link rel="dns-prefetch" href="https://images.unsplash.com">
```
